# Airbag_ECU_FunctionalSafety_SoSe2026
Functional Safety SoSe26 : Project work on Hazard and Risk assessment to obtain an ASIL level for the Airbag ECU System.
A structured, engineering-focused project dedicated to analyzing and ensuring the functional safety of an Airbag Electronic Control Unit (ECU) in accordance with ISO 26262 and IEC 61508.
